# Student-Mnanagement-SQL-
This project is a simple Student Management System database built using MySQL. 
It manages students, teachers, courses, enrollments, and attendance records with proper relations.

## Features
- Store student & teacher information
- Manage multiple courses
- Many-to-many relationship using enrollments
- Attendance tracking
- Useful analytics queries

## How to Run
1. Open MySQL Workbench / XAMPP MySQL
2. Run `student_management_system.sql`
3. Check example queries at the bottom
